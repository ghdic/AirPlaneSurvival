# AirPlaneSurvival
My unity game



# Game Mode

## Solo Survival Mode

* 최대한 오래 살아남는것을 목표로 한다.
* 필드 제한 없음
* 주기적으로 적 추가 생성

## Multi Copperation Mode

* 아군이 있으며 아군 비행기는 항상 10마리로 고정한다(아군AI수는 (10 -MultiPlayer))
* 필드 크기 제한 있음
* 정해진 수의 적을 다 죽이면 Clear
* WayPoint의 일부는 각 MultiPlayer의 포지션으로 설정한다.
* 네트워크는 PhtonClude를 이용한다.
